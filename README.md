## yiya

**I build verification-first tools — for code, and for the humans using them.**
**Agents are cheap; proof is the product.**

Non-traditional engineer in San Francisco. As AI takes over more of the execution, the scarce, defensible asset is *proof* — that the code is correct, and that the human is too. That one bet is the spine of everything below: founder of **Coaur** (the lab), building **[Synapi](https://synapi.app)** (the product), and shipping the agentic toolchain underneath in the open.

---

**Verify the work — code that proves itself**
- [**claude-code-skills**](https://github.com/yiyaw-lab/claude-code-skills) — 17 Claude Code skills built to make an agent *prove* its work, not describe it. First overnight run: 6/6 tasks shipped at **$11.52 per shipped task** — and the one defect was caught when an adversarial reviewer refused an agent that had gamed its own done-check.
- [**skarf**](https://github.com/yiyaw-lab/skarf) — source-backed memory for what AI chats leave behind.

**Verify the human — minds that stay sharp as the tools get smarter**
- [**Synapi**](https://synapi.app) — the metacognitive layer that proves *you're* getting smarter, not just your tools: spaced retrieval, Bayesian mastery, a portable Learner Card.
- [**ocii**](https://github.com/yiyaw-lab/ocii) — open infrastructure for evaluating genuine human understanding vs fluent AI-assisted output.

**The factory that builds itself — a toolchain that compounds**
- [**buildlog**](https://github.com/yiyaw-lab/buildlog) — a local continuity CLI for builders who ship with agents. The skills suite grades and sharpens *itself* (harvest → hone → grade-session), getting better from its own report cards.
