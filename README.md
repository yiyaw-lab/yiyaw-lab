## yiya

**I build verification-first tools — for code, and for the humans using them.**
**Agents are cheap; proof is the product.**

Non-traditional engineer in San Francisco. As AI takes over execution, the scarce and defensible asset is *proof* — that the code is correct, and that the human is too. That one bet is the spine of everything I build: founder of **Coaur** (the lab), building **[Synapi](https://synapi.app)** (the product), and shipping the agentic toolchain underneath in the open.

---

**Verify the work — code that proves itself**
- [**agent-armor**](https://github.com/yiyaw-lab/agent-armor) — armor for coding agents: a playbook of Claude Code skills that make an agent *prove* its work, not describe it. First overnight run shipped 6/6 tasks at **$11.52 each**, and the one defect of the night was caught when an adversarial reviewer refused an agent that had gamed its own done-check.
- [**skarf**](https://github.com/yiyaw-lab/skarf) — source-backed memory for what AI chats leave behind.

**Verify the human — minds that stay sharp as the tools get smarter**
- [**Synapi**](https://synapi.app) — the metacognitive layer that proves *you're* getting smarter, not just your tools: spaced retrieval, Bayesian mastery, a portable Learner Card.
- [**ocii**](https://github.com/yiyaw-lab/ocii) — open infrastructure for evaluating genuine human understanding vs fluent AI-assisted output.

**A toolchain that compounds — the factory that builds itself**
- [**buildlog**](https://github.com/yiyaw-lab/buildlog) — a local continuity CLI for builders who ship with agents: each session starts where the last one ended. Paired with agent-armor, whose suite grades and sharpens *itself* from its own report cards (harvest → grade-session → hone).
